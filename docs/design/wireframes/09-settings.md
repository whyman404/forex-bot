## Design notes

- "Done" button on API key reveal requires explicit click вҖ” user must acknowledge they saved the key
- Danger Zone (delete account) is at bottom of Account tab вҖ” lowest visual prominence, hard to reach accidentally
- 2FA disable requires both password AND 2FA code вҖ” two-factor to remove two-factor
- API keys tab only appears for Lifetime plan вҖ” not hidden for Free/Pro, but shows upgrade CTA instead
- Notification defaults: system-critical notifications (broker disconnected, new login) default to Email ON
- Session revoke: "this session" row cannot have a revoke button вҖ” only other sessions
- All destructive actions (revoke session, revoke key, delete account) have confirmation modals
